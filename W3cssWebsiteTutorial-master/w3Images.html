Rounded Image

The w3-round class adds rounded corners to an image:

Example

<img src="a.jpg" class="w3-round" alt="Norway">

Circled Image

The w3-circle class shapes an image to a circle:

Example

<img src="a.jpg" class="w3-circle" alt="Alps">

Bordered Image

The w3-border class adds borders around the image:

Example

<img src="b.jpg" class="w3-border w3-padding" alt="Alps">

Image as a Card

Wrap any of the w3-card-* classes around the <img> element to display it as a card (add shadows):


Simon

The boss of all bosses

Example

<div class="w3-card-4">
  <img src="avatar.png" alt="Person">
</div>

Opacity

The w3-opacity classes make images transparent:

Normal

w3-opacity-min

w3-opacity

w3-opacity-max

Example

<img src="d.jpg" alt="Forest" class="w3-opacity-min">
<img src="d.jpg" alt="Forest" class="w3-opacity">
<img src="d.jpg" alt="Forest" class="w3-opacity-max">

Grayscale

The w3-grayscale classes add a grayscale effect to an image:

Normal

w3-grayscale-min

w3-grayscale

w3-grayscale-max

Example

<img src="d.jpg" alt="Table" class="w3-grayscale-min">
<img src="d.jpg" alt="Table" class="w3-grayscale">
<img src="d.jpg" alt="Table" class="w3-grayscale-max">

Note: The w3-grayscale classes are not supported in IE 11 and earlier versions.

Sepia

The w3-sepia classes add a sepia effect to an image:

Normal

w3-sepia-min

w3-sepia

w3-sepia-max

Example

<img src="d.jpg" alt="Table" class="w3-sepia-min">
<img src="d.jpg" alt="Table" class="w3-sepia">
<img src="d.jpg" alt="Table" class="w3-sepia-max">

Note: The w3-sepia classes are not supported in IE 11 and earlier versions.

Hover Effects

You can also add special effects on hover/mouse-over.

w3-hover-opacity

w3-hover-grayscale

w3-hover-sepia

Example

<img src="d.jpg" alt="Einstein" class="w3-hover-opacity">
<img src="image.jpg" alt="Einstein" class="w3-hover-grayscale">
<img src="d.jpg" alt="Einstein" class="w3-hover-sepia">

Opacity Off

Add transparency on hover:

Remove transparency on hover:

The w3-hover-opacity class adds transparency to the image on mouse-over, and the w3-hover-opacity-off class removes transparency on mouse-over.

Example

<img src="d.jpg" class="w3-hover-opacity" alt="Alps">
<img src="d.jpg" class="w3-opacity w3-hover-opacity-off" alt="Alps">
