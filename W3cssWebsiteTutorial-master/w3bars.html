W3.CSS Bars

Horizontal Bars

Horizontal bars are common web design elements.
The w3-bar class is used to style an horizontal bar.

Example :

<div class="w3-bar w3-green">
  <div class="w3-bar-item">India</div>
  <div class="w3-bar-item">China</div>
  <div class="w3-bar-item">Africa</div>
</div>

Output :

India
China
Africa

Vertical Bars

Vertical bars (sidebars) are also common in web design.
The w3-bar-block class is used to style a vertical bar.

Example :

<div class="w3-bar-block w3-green" style="width:100px">
  <div class="w3-bar-item">India</div>
  <div class="w3-bar-item">China</div>
  <div class="w3-bar-item">Africa</div>
</div>

Output :

India
China
Africa

Hover Colors

You can use any hover color to style a bar.
Mouse over the bar items to see the effect.

Example :

<div class="w3-bar w3-black">
  <div class="w3-bar-item w3-hover-red">India</div>
  <div class="w3-bar-item w3-hover-blue">China</div>
  <div class="w3-bar-item w3-hover-green">Africa</div>
</div>

Output :

India
China
Africa

Bar Links

Providing navigation is a typical use for bars.

Example :

<div class="w3-bar w3-black">
  <a href="#" class="w3-bar-item w3-hover-green">India</a>
  <a href="#" class="w3-bar-item w3-hover-green">China</a>
  <a href="#" class="w3-bar-item w3-hover-green">Africa</a>
</div>

Output :

India China Africa

Bar Buttons

The w3-button class is perfect for styling links in a bar.
Mouse over the bar items to see the effect.

Example :

<div class="w3-bar w3-black">
  <a href="#" class="w3-bar-item w3-button">India</a>
  <a href="#" class="w3-bar-item w3-button">China</a>
  <a href="#" class="w3-bar-item w3-button">Africa</a>
</div>

Output :

India China Africa

Responsive Bar

The w3-mobile class is perfect for making bar items responsive.
Resize the window to see the effect.

Example :

<div class="w3-bar w3-black">
  <a href="#" class="w3-bar-item w3-button w3-mobile">India</a>
  <a href="#" class="w3-bar-item w3-button w3-mobile">China</a>
  <a href="#" class="w3-bar-item w3-button w3-mobile">Africa</a>
</div>

Output :

India China Africa

Right-Aligned Bar Items

The w3-right class is perfect for making bar items right-aligned.

Example :

<div class="w3-bar w3-black">
  <a href="#" class="w3-bar-item w3-button">India</a>
  <a href="#" class="w3-bar-item w3-button">China</a>
  <a href="#" class="w3-bar-item w3-button w3-right">Africa</a>
</div>

Output :

India China Africa
